# 一方天地 · 使用手册

这是你的个人网站(Astro 版)。设计、动画、配色都已焊死在框架里,
你**永远不用碰 HTML/CSS**,只需改下面几个文件就能更新内容。

---

## 一、本地预览(边改边看)

在 `site` 目录里打开终端,跑:

```
npm run dev
```

浏览器打开提示的地址(通常 http://localhost:4321),
之后你**改任何文件,浏览器自动刷新**,所见即所得。

按 `Ctrl + C` 停止预览。

---

## 二、怎么改内容

### 1. 发一篇新博文

在 `src/content/blog/` 新建一个 `.md` 文件(文件名随意,英文最好),
开头照抄这个格式:

```markdown
---
title: "文章标题"
date: 2026-06-12
tags: ["随笔", "摄影"]
excerpt: "列表里显示的一句话摘要。"
foot: "结尾的斜体小字(可选,不要就删掉这行)"
---

正文从这里开始,直接写。

空一行就是新段落。**这样**是加粗。

想分点就用:
- 第一点
- 第二点
```

存盘,网站自动多出这篇文章,排版、展开动画全自动套好。

### 2. 改「关于」的文字

打开 `src/data/profile.ts`,改 `about` 那几段文字。

### 3. 改名字 / 头衔 / 社交链接 / 默认配色

打开 `src/data/profile.ts`:
- `name` `title` `tagline` —— 左上角的名字、头衔、简介
- `socials` —— GitHub / Instagram / 邮箱,**留空字符串 `''` 该图标就不显示**
- `defaultPalette` —— 首次访问的默认配色(amber/pine/terra/indigo/heather/ink/stone/olive/onyx/claude/openai)
- `defaultTheme` —— 默认明暗(dark / light)

### 4. 加 / 改「项目」卡片

打开 `src/data/projects.ts`,复制一个 `{ ... }` 块改掉内容。
- `repo` 填 `"用户名/仓库名"` → 自动显示实时 star 数(随你 GitHub 变化)
- `logo` 填图片路径(图片放 `public/assets/` 里),留空则显示项目名

### 5. 加 / 改「站点」卡片

打开 `src/data/sites.ts`,复制一个 `{ ... }` 块改掉内容。用法同项目。

### 6. 加 / 改「历程」

打开 `src/data/experiences.ts`,复制一个 `{ ... }` 块改掉内容。

### 7. 换图片

把图片丢进 `public/assets/` 文件夹,
然后在数据文件里引用 `/assets/你的图片名`。
